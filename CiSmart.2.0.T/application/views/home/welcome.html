Welcome to CiSmart

CiSmart adalah hasil dari modifikasi PHP Web Framework dengan core system menggunakan CodeIgniter 2.0 dan template engine smarty 3.0.7.
Framework ini dimodifikasi sedemikan rupa sesuai dengan kebutuhan pengembangan software berbasis web.
Modifikasi mencakup development environment standards sehingga mempersatukan setiap programmmer yang terlibat untuk menggunakan standard yang sama dalam melakukan pengembangan web didalam aturan - aturan dasar CI dan Smarty sendiri.
Didalam standarisasi ini juga telah disediakan banyak contoh (templates) coding standards yang meliputi :

  • CRUD (Create, Read, Update and Delete) Coding Standards
  • Styles Templating
  • Javascript Interactive
